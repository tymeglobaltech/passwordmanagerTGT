Create Backup:

  sudo mkdir -p ~/backups
  sudo nano ~/backup-db.sh

			#!/bin/bash
			BACKUP_DIR="/home/mgeorgiades/backups"
			TIMESTAMP=$(date +%Y%m%d_%H%M%S)
			FILENAME="passwordpal_${TIMESTAMP}.sql.gz"

			pg_dump -U passwordpal passwordpal | gzip > "${BACKUP_DIR}/${FILENAME}"

			# Keep only last 30 days of backups
			find "${BACKUP_DIR}" -name "passwordpal_*.sql.gz" -mtime +30 -delete

			echo "Backup completed: ${FILENAME}"

  chmod +x /home/mgeorgiades/backup-db.sh


Schedule it to run daily via cron:

  crontab -e

Add:
  0 2 * * * /home/mgeorgiades/backup-db.sh



restore backup:
gunzip -c /home/mgeorgiades/backups/passwordpal_XXXXXXXX.sql.gz | psql -U passwordpal passwordpal
